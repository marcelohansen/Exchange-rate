# Exchange-rate
Dashboard with historical USD/BRL exchange rate data from the Banco Central do Brasil.

https://bit.ly/CotacaoUSD

This dashboard was built with two APIs:

- 1️⃣ Collect historical data of the USD/BRL exchange rate from the Central Bank of Brazil
- 2️⃣ Collect the USD/BRL and EUR/BRL exchange rate "in real time" from awesomeapi

Customer uses Portuguese (BRL) and needs four decimal places.

The dashboard has simple and basic design features with some date filters.
